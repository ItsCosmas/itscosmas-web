### Running Locally

Clone this repository, install the dependencies and devDependencies, and start the server.

```sh
$ git clone https://github.com/ItsCosmas/itscosmas.git
$ cd itscosmas
$ npm install
$ npm start
```

To build for production ...

```sh
$ npm build
```
